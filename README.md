# Credit Scoring Model

## Overview
The Credit Scoring Model is a Machine Learning web application developed using Python and Flask. It predicts the credit score, loan approval status, and risk level of an applicant based on financial details entered by the user.

## Features
- Predicts credit score
- Determines loan approval or rejection
- Displays risk level
- User-friendly web interface
- Built using Machine Learning (Random Forest Classifier)

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Pandas
- NumPy
- Scikit-learn

## Input Parameters
- Age
- Monthly Income
- Loan Amount
- Credit History

## Output
- Credit Score
- Risk Level
- Loan Approval Status

## Installation

1. Clone the repository:
```bash
git clone https://github.com/choudarysurakshitha/Credit-Scoring-Model.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and visit:
```
http://127.0.0.1:5000
```
